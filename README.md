The Register Page allows new users to create an account by entering basic details like name, email, and password, enabling them to access the Shoes App features.

Login page

<img width="484" height="959" alt="Screenshot (164)" src="https://github.com/user-attachments/assets/38c2e3ff-030f-4ba9-ab7e-7c18c4ccf944" />

The Login Page allows users to access the Shoes App by entering their email and password. It verifies credentials and provides secure access to features like browsing shoes, viewing details, and managing the wishlist or cart.

Home page

<img width="483" height="966" alt="Screenshot (168)" src="https://github.com/user-attachments/assets/0754b626-314d-4313-a5b8-a23eca989235" />

The Home Screen is the main interface of the Shoes App, featuring an App Bar, search bar, and dark mode option. It displays all shoes in a grid/list format, allowing users to easily browse, search, switch themes, and view product details for a smooth shopping experience.

Drawer page

<img width="489" height="969" alt="Screenshot (169)" src="https://github.com/user-attachments/assets/2618e3ab-4f62-44a2-90a3-d05146ac732a" />

The drawer page provides easy navigation to key sections like Home, Favorites, and Cart, allowing users to quickly access content, saved items, and selected products in a simple and user-friendly way

Favorite page

<img width="490" height="954" alt="Screenshot (170)" src="https://github.com/user-attachments/assets/50dbbf86-5885-4bd2-aa12-79e25163f975" />

The Wishlist Screen lets users save their favorite shoes for later. They can add items from the Home or Product Detail Screen, view all saved products in one place, and easily remove them or move them to the cart for purchase.

Detail page

<img width="487" height="969" alt="Screenshot (167)" src="https://github.com/user-attachments/assets/dc104aad-b12d-4477-a6cc-f3651239c1d1" />

The Product Detail Screen shows complete information about a selected shoe, including images, colors, sizes, and variations. Users can choose options and add the item to the cart or wishlist, while also viewing details like price, material, and style to make better decisions.

Cart page

<img width="487" height="977" alt="Screenshot (171)" src="https://github.com/user-attachments/assets/05032306-3e5a-4f4f-9805-e8ce1c6804fd" />

The Cart Screen lets users view all selected shoes for purchase in one place. It shows details like name, price, and quantity, and allows users to update or remove items. Users can also proceed to checkout to complete their order.

Order confirm page
